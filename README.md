# Development of a Smart Indoor Environment Alert System Using Raspberry Pi

AIoT설계입문 기말 프로젝트 (202578207 황사익)

가스 누출과 외부 미세먼지/날씨 정보를 종합하여 실내 환경 위험을 자동으로 알려주는 라즈베리파이 기반 IoT 시스템입니다.

----------------------------------------

##  프로젝트 소개

가스가 새면 즉시 환기하라고 알려주고, 미세먼지가 나쁘거나 비/눈이 오면 창문을 닫으라고 알려줍니다.
실내외 공기질 정보를 사람이 직접 비교 판단하지 않아도, 시스템이 자동으로 가장 안전한 행동을 LED 색깔로 알려주도록 설계했습니다.

가스 누출처럼 즉각적인 위험과 미세먼지처럼 장기적인 위험을 우선순위로 구분하여 처리합니다.

----------------------------------------

##  동작 원리

🔴 빨강 LED = 가스 감지 (최우선 경보, 가스센서 감지 시 즉시 작동)  
🟢 초록 LED = 환경 나쁨 (창문 닫기, 미세먼지(PM2.5) 나쁨 또는 비/눈 올 때)  
🔵 파랑 LED = 시스템 정상 작동 중 (전원 켜지면 항상 ON)  

가스 감지 시 다른 조건과 상관없이 빨강이 최우선으로 동작합니다.

----------------------------------------

##  사용 부품

- Raspberry Pi 5
- MQ-2 가스/연기 감지 센서 (5V)
- 능동 부저 (3.3V)
- LED 3개 (빨강 / 파랑 / 초록)
- 330Ω 저항 ×3
- 브래드보드, 점퍼케이블

----------------------------------------

##  회로 연결

가스센서 VCC → 4번 핀 (5V)
가스센서 GND → 6번 핀 (GND)
가스센서 DOUT → GPIO 17 (11번 핀)
부저 VCC → 17번 핀 (3.3V) 5V 아님
부저 GND → 9번 핀 (GND)
부저 DOUT → GPIO 18 (12번 핀)
초록 LED → GPIO 16 (36번 핀)
파랑 LED → GPIO 20 (38번 핀)
빨강 LED → GPIO 21 (40번 핀)

----------------------------------------

##  활용한 수업 내용

##  파일 구조

main_system.py        → 센서+API+LED+부저 단독 실행 버전
app_server.py          → Flask 웹서버 버전 (메인)
templates/index.html   → 웹 대시보드 페이지

----------------------------------------

##  실행 방법

sudo apt-get install python3-flask

app_server.py 안의 API_KEY를 본인 OpenWeatherMap 키로 변경 후 실행

sudo python3 app_server.py

실행 후 브라우저에서 라즈베리파이 IP 주소로 접속하면 실시간 상태를 확인할 수 있습니다.

- 경보음 끄기 버튼 (다음 새 이벤트 발생 전까지 음소거)
